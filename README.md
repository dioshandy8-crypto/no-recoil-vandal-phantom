# Valorant Recoil Assist (Python)

Eksperimen kontrol mouse berdasarkan deteksi warna untuk membantu kompensasi recoil pada senjata Vandal dan Phantom.

## Fitur
- **Color-Based Activation**: Hanya aktif jika kursor mengenai musuh (Purple Tritanopia).
- **Weapon Profiles**: Hotkey untuk berpindah antara Vandal (F1) dan Phantom (F2).
- **Performance**: Menggunakan Win32API untuk input mouse yang cepat.

## Cara Instalasi
1. Clone repo ini: `git clone https://github.com`
2. Masuk ke folder: `cd NAMA_REPO`
3. Install dependensi: `pip install -r requirements.txt`
4. Jalankan: `python main.py`

## Pengaturan Game
- **Enemy Highlight Color**: Purple (Tritanopia).
- **Display Mode**: Windowed Fullscreen.

## Disclaimer
Proyek ini hanya untuk tujuan edukasi. Penggunaan di game online berisiko ban permanen.
